## Installation & Usage

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Train a model:**
   ```bash
   python src/train.py
   ```

3. **Run inference:**
   ```bash
   python src/inference.py --model_path src/best_model.npy
   ```
